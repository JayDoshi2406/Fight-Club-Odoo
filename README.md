# Fight Club – Fleet Management System

##  Overview

Fight Club is a Fleet Management System designed to manage transportation operations efficiently. The system helps administrators manage vehicles, drivers, trips, fuel expenses, and maintenance records while providing insights into fleet performance and operational costs.

The project is built using **Node.js and Express.js** with a modular architecture to ensure scalability and maintainability.

---

##  Features

* Driver management
* Vehicle registration and tracking
* Trip management system
* Fuel expense tracking
* Vehicle maintenance logs
* Analytics and reporting
* Secure backend middleware

---

##  Tech Stack

* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Architecture:** MVC (Model-View-Controller)
* **File Uploads:** Multer

---

##  Project Structure

```
backendNode/
```

##  Installation

### 1 Clone the repository

```
git clone https://github.com/JayDoshi2406/Fight-Club-Odoo.git
```

### 2️ Navigate to project directory

```
cd Fight-Club-Odoo
```

### 3️ Install dependencies

```
npm install
```

### 4️ Run the server

```
node app.js
```
